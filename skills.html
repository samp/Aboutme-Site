CSS & ECMAscript

CSS stands for Cascading Style Sheets and describes how HTML is displayed in a web browser. CSS allows styles and content to be separated, improving accessibility, flexibility, and control. It is used to alter the appearance of a web page. The CSS examples below demonstrate how CSS can be used to completely change the way a page looks - both examples have identical HTML.

ECMAscript is a specification used to standardise JavaScript. It is used for both client-side web scripting and server applications using Node.js. Commonly, it allows webpages to be interactive, including storing data. The example below demonstates interactivity, storage, and creating elements from a file.
